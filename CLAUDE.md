# Project: Medical App Security Demo

## Purpose
Educational security demo for Web + Mobile API vulnerabilities.
Target: OWASP Top 10, focused on authentication bypass first.

## Stack
- Backend: Python / FastAPI
- Attack scripts: Python
- Language: English (all code comments and docs)

## Rules
- vulnerable/ and secure/ must always be paired
- Never use real patient data, use mock data only
- All filenames and comments must be in English
