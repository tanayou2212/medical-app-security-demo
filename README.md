# medical-app-security-demo

A hands-on security demonstration project for medical device software, covering common vulnerabilities, exploit techniques, and secure coding practices relevant to FDA, EU MDR, and MHLW regulatory requirements.

## Structure

| Directory | Description |
|-----------|-------------|
| `vulnerable/` | Intentionally vulnerable sample applications |
| `attack/` | Proof-of-concept exploits (authorized use only) |
| `secure/` | Remediated, hardened versions |
| `docs/` | Threat models, security analyses, compliance notes |

## Disclaimer

All code in this repository is for **educational and authorized security testing purposes only**.
Do not deploy vulnerable code in production environments or against systems you do not own.
